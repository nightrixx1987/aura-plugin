AURA - Professional Parametric EQ
by Unproved Audio
=====================================

Thank you for choosing Aura!

SYSTEM REQUIREMENTS
-------------------
Windows:
- Windows 10/11 (64-bit)
- VST3 compatible DAW (Ableton Live, FL Studio, Cubase, etc.)
- 4 GB RAM minimum
- Screen resolution: 1280x720 or higher

macOS:
- macOS 10.15 (Catalina) or later
- Intel (x86_64) or Apple Silicon (arm64)
- VST3 compatible DAW
- 4 GB RAM minimum

INSTALLATION OPTIONS
--------------------
1. STANDALONE APPLICATION
   Runs Aura as an independent application for testing and practice.
   Windows: Installs to Program Files\Unproved Audio\Aura\
   macOS:   Installs to /Applications/

2. VST3 PLUGIN
   Installs the plugin to your VST3 folder. Your DAW should automatically
   detect the plugin after installation.

DEFAULT VST3 LOCATIONS
----------------------
Windows: C:\Program Files\Common Files\VST3\
macOS:   /Library/Audio/Plug-Ins/VST3/

If your DAW uses a different VST3 folder, you can change this during
installation (Windows) or manually move the .vst3 bundle.

AFTER INSTALLATION
------------------
1. Start your DAW
2. Rescan your VST3 plugins if necessary
3. Look for "Aura" in your plugin list under "Unproved Audio"

UNINSTALLING
------------
Windows: Use "Add/Remove Programs" or the Uninstaller in Start Menu.
macOS:   Run the included uninstall.sh script, or manually delete:
         - /Applications/Aura.app
         - /Library/Audio/Plug-Ins/VST3/Aura.vst3
